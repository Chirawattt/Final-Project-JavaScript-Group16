## 📘 About This Project

This Android application was developed as an educational tool to help users learn the **JavaScript** programming language in an engaging and interactive way.

The application provides:

- 🧠 **Lessons**: Step-by-step conceptual explanations with examples.
- ✅ **Quizzes**: Interactive multiple-choice questions to reinforce learning.
- 💻 **Practice Mode**: Hands-on coding with live execution and feedback.
- 🎉 **Progress Tracking**: Users can track their progress and achievements via visual indicators, sounds, and animations.

---

## 🛠️ Tech Stack

- `Java` + `XML` – Native Android development  
- `WebView` – For JavaScript code execution  
- `SharedPreferences` – For storing user progress  
- `Lottie Animations` – For visual feedback and fun  
- `Custom UI Components` – Designed for clarity and accessibility  
- Fully **offline** – No backend required

1. Clone this repository.

4. Enjoy learning JavaScript interactively!
